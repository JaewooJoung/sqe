Key Statistics & Observations

Most Effective Programs

  • Finland: 5.7% re-offense vs 30% without program
  • US: Could save 800 lives/year with full implementation
  • Denmark: Improved from 24 to 450 participants with program changes

Cost Ranges

  • Highest: Belgium (€3,500), Austria (€3,400+/year)
  • Mid-range: Finland (€1,440-2,400), France (€1,300)
  • US: $70-150 install + $60-80/month

Note: This table is based on data available in the document, which may not reflect the most current status for all countries. Some programs have evolved since the data collection date.
